# Kişisel Finans Takip Sistemi (Personal Finance Tracking System)

## Proje Özeti
Yazılım Mühendisliği ders projesi - ailelerin ortak finansal yönetimi için web uygulaması.

## Teknoloji Yığını
- **Frontend:** React (JavaScript)
- **Backend:** Python + Flask
- **Veritabanı:** SQL Server

## Kullanıcı Rolleri
1. **Individual:** Kişisel harcama yönetimi
2. **Family Leader:** Aile bütçesi yönetimi, alt üye davetleri
3. **Admin:** Sistem yönetimi, loglama, yedekleme

## Temel Özellikler
- ✓ Güvenli kimlik doğrulama (JWT + bcrypt)
- ✓ Harcama/Gelir CRUD işlemleri
- ✓ Kategori ve etiket sistemi
- ✓ Bütçe alarmları (%80 bildirimi)
- ✓ Aylık/Yıllık raporlama ve grafikler
- ✓ Ortak bütçe yönetimi
- ✓ Admin loglama ve yedekleme

## Dosya Yapısı
```
FİNANS TAKİP/
├── backend/          (Python Flask API)
├── frontend/         (React SPA)
└── database/         (SQL Server schema)
```
